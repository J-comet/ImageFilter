# ImageFilter
원하는 이미지를 선택해서 필터를 적용시킨 후 지인들에게 공유할 수 있는 앱 입니다. 

### 사용한 것
- Koin
- ViewBinding
- ViewModel
- Round Image
- GPU Image

### 미리보기
![Webp net-gifmaker](https://user-images.githubusercontent.com/67407666/89728216-9a643800-da66-11ea-8ca4-8fe52974885b.gif)


### 주요기능

- 이미지 필터
- 필터 적용 전·후 이미지 비교
- 공유하기
- 이미지 필터 적용했던 이미지들 모아보기
